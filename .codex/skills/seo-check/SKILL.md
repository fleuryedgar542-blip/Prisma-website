# SEO Check

Gebruik deze skill wanneer pagina's zijn aangepast.

## Doel
Controleer de basis SEO van aangepaste pagina's.

## Controle

Controleer per pagina:

- title aanwezig
- meta description aanwezig
- canonical aanwezig
- hreflang aanwezig
- exact 1 H1
- OG tags aanwezig
- Twitter tags aanwezig
- structured data aanwezig

## Interne links

Controleer:

- navigatie links
- footer links
- CTA links
- blog links
- tool links

Geen kapotte links.

## Productnamen

Gebruik exact:

PRISMA Dag  
PRISMA Week  
PRISMA Signaal  

PRISMA Day  
PRISMA Week  
PRISMA Signal  

Gebruik dagplanner of weekplanner niet als productnaam.

## Metadata taal

Controleer dat:

- NL pagina Nederlandse metadata heeft
- EN pagina Engelse metadata heeft

## Eindcontrole

Controleer:

- geen dubbele H1
- correcte canonical
- correcte hreflang
- werkende interne links
