# NL EN Consistency

Gebruik deze skill wanneer een pagina in zowel Nederlands als Engels bestaat.

## Doel

Zorg dat NL en EN pagina's structureel overeenkomen.

## Nederlandse pagina's zijn leidend voor

- layout
- CSS
- sectievolgorde
- wrappers
- navigatie
- footer
- CTA blokken

## Engelse pagina's moeten

- dezelfde structuur hebben
- dezelfde layout hebben
- Engelse tekst tonen
- eigen metadata hebben

## Vergelijk

Controleer:

- hero
- secties
- CTA's
- navigatie
- footer
- toolstructuur

## Productnamen

Gebruik exact:

PRISMA Dag  
PRISMA Week  
PRISMA Signaal  

PRISMA Day  
PRISMA Week  
PRISMA Signal  

Gebruik dagplanner of weekplanner niet als productnaam.

## Taal

NL pagina = Nederlandse tekst  
EN pagina = Engelse tekst  

Geen mix.

## Toolpagina's

Controleer extra:

- PRISMA Dag ↔ PRISMA Day
- PRISMA Week ↔ PRISMA Week
- PRISMA Signaal ↔ PRISMA Signal

Controleer:

- labels
- knoppen
- headings
- analyseblokken
- layout

## Eindcontrole

Controleer:

- interne links
- taalwissels
- layout consistentie
- productnamen
